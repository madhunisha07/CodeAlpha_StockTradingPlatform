This project is a Java-based Stock Trading Platform developed as part of the CodeAlpha Internship Program.
It simulates a basic stock trading environment where users can view market stocks, buy and sell shares, and track their portfolio performance using Object-Oriented Programming (OOP) concepts.
Features
	•	View available market stocks with prices
	•	Buy stocks by entering stock symbol and quantity
	•	Sell owned stocks
	•	View user portfolio and total investment value
	•	Menu-driven console interface
	•	Implements core OOP principles
	•	Uses Java collections for dynamic data handling
  Technologies Used
	•	Java
	•	Object-Oriented Programming (OOP)
	•	Scanner (User Input)
	•	HashMap / ArrayList
